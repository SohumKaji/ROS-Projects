>> This ROS project 
- transforms co-ordinates between base coordinates and robot coordinates
- transforms co-ordinates between base coordinates and object coordinates 
- transforms co-ordinates between robot coordinates and camera coordinates
	using the object as a reference ("looking" at the object)

>>The code that I worked on can be found in:
- catkin_ws/src/project2_solution/scripts/solution.py


To setup/run project for yourself:
Run: 
source setup_project2.sh
rosrun project2_solution solution.py
Check canvas for a visual representation
